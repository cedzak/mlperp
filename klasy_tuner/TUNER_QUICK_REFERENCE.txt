┌─────────────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                                  │
└─────────────────────────────────────────────────────────────────────┘

Problem: Out of Memory (OOM)
  ✓ BATCHSIZE = 64 (↓ z 128)
  ✓ SEQLEN = 48 (↓ z 72)
  ✓ Zmniejsz max jednostek w warstwach

Problem: Za długo trwa
  ✓ MAX_TRIALS = 20 (↓ z 50)
  ✓ ILERUNOW_PER_TRIAL = 1 (↓ z 3)
  ✓ EPOCHS = 150 (↓ z 200)
  ✓ Użyj "hyperband"

Problem: Nie poprawia baseline
  ✓ MAX_TRIALS = 100 (↑)
  ✓ Zmień strategię na "hyperband"
  ✓ Sprawdź przestrzeń przeszukiwania

Problem: Kernel died / GPU crash
  ✓ Zmniejsz BATCHSIZE
  ✓ Sprawdź temperatury GPU
  ✓ Dodaj więcej clear_session()
